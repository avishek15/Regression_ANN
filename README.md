# Regression ANN — Neural Networks for Regression

**Artificial neural network implementations for regression tasks.**

## What It Does

Implements neural networks for predicting continuous values. Covers basic to advanced architectures.

**Why it matters:** Regression is fundamental to ML. Neural networks handle non-linear relationships that traditional methods can't.

## Features

- **Multiple architectures** — MLP, Deep networks
- **Custom activation functions**
- **Regularization** — L1, L2, Dropout
- **Hyperparameter tuning** — Grid search examples

## Quick Start

\`\`\`bash
# Clone
git clone https://github.com/avishek15/Regression_ANN.git
cd Regression_ANN

# Install
pip install -r requirements.txt

# Train
python train.py --dataset boston --epochs 100
\`\`\`

## Tech Stack

- **Python 3.8+**
- **TensorFlow/Keras** — Deep learning
- **Scikit-learn** — Preprocessing
